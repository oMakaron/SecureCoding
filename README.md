## Guide for git  
###  Starting 
Every session - always get the current version of main:
- git checkout main
- git pull origin main
- git checkout [your-branch]
- git merge main

### During local development 
- git status *just to make sure your on your branch*
- git add [file]
-  git commit -m "A breif description on what you fixed, dont do "Fixed things in main""

### When finished coding session locally 
- git push origin [your-branch] 

### Pushing to main 
 - push to your branch
 - Go to github and make a pull request.
